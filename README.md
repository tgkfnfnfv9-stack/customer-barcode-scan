# お客様用 バーコードスキャン｜株式会社小林機械

機械の銘板バーコードを iPhone で読み取り、公開サイトの機械ページへ誘導するお客様向けツールです。
ログインは不要。読み取りエンジンは端末内で完結（オフライン動作）、単一の `index.html` だけで動きます。

**公開URL：** https://tgkfnfnfv9-stack.github.io/リポジトリ名/
（↑ 実際のURLに書き換えてください）

---

## 使い方

1. 上記URLを iPhone の Safari で開く（ホーム画面に追加すると便利）
2. 「撮影して読み取る」で機械の銘板のバーコードを撮影
3. 読み取ったコードが履歴に追加される（最新が一番上）
4. 各行の「この機械を見る」を押すと、そのコードで公開サイトを検索し、該当機械のページが開く

### 主な機能

- 読み取り履歴が積み上がる（同じコードは重複せず先頭へ移動）
- 各コードから「この機械を見る」で公開サイトの機械ページへ移動
- カメラが使えないときは番号を直接入力して登録することも可能
- 個別コピー（各行）／すべてコピー（まとめて）／個別削除（✕）／すべてリセット
- 履歴は端末内に保存（閉じても更新しても残る）

## 更新方法

`index.html` を新しい版で上書きコミットするだけで、数分後に公開URLへ自動反映されます（URLは変わりません）。

1. リポジトリの `index.html` を開く
2. 鉛筆（Edit）または「Upload files」で新しい版に差し替え
3. 「Commit changes（変更をコミット）」で保存

## 技術メモ

- 単一 HTML（CDN・ネット依存なし）。読み取りエンジンは ZXing-C++（WASM）を base64 で埋め込み
- 遷移先（公開サイトの商品検索 → 機械詳細ページ）
  - `https://www.kkmt.co.jp/products?utf8=✓&search_forms_products_search[keyword]=<コード>&commit=検索`
  - 例) `P009000` → 検索 → `https://www.kkmt.co.jp/machines/159211`
- 印字と符号化値が異なる機械は、`index.html` 内の `OVERRIDE_MAP` に登録して対応
